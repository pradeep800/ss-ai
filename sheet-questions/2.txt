You are given an integer ‘N’. You need to return the first ‘N’ rows of Pascal’s triangle.
Example:
Input:
N = 4
Output:
1
1 1
1 2 1
1 3 3 1
Explanation: The output matrix has the first four rows of Pascal’s Triangle.
